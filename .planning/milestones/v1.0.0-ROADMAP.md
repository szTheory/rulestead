# Milestone v1.0.0: General Availability (GA)

**Status:** ✅ SHIPPED 2026-05-21
**Phases:** 26-28
**Total Plans:** 11

## Overview

`v1.0.0` closes the general-availability milestone for Rulestead. This release freezes the public API boundary, hardens authorization around a canonical three-role RBAC model, and proves the full stack through a Docker-backed Phoenix + Next.js demo environment that exercises the mounted admin, runtime bridge, and OpenFeature integration end to end.

## Milestone Summary

**Key Decisions:**

- Accepted one documented tooling override for `API-02` rather than reopening product code for the known `rulestead_admin` Dialyxir/Erlang warning baseline.
- Preserved the host-owned policy seam as the RBAC integration contract instead of introducing a third-party authorization layer.
- Kept the GA demo boundary inside `examples/demo/**` so the shipped product remains the linked-version, two-package monorepo.

**Issues Resolved:**

- Public/private API packaging is now explicit in HexDocs and code-level module visibility.
- Mounted admin authorization now derives from backend capability truth instead of route-local heuristics.
- The GA demo stack is proven through Compose smoke and Playwright browser proof on 2026-05-21.

**Issues Deferred:**

- Phase 25 tenancy helpers remain a post-GA candidate rather than part of the shipped GA milestone.

**Technical Debt Incurred:**

- `API-02` is satisfied with a documented tooling override because `rulestead_admin` still carries the known unignorable Dialyxir/Erlang baseline warning.
