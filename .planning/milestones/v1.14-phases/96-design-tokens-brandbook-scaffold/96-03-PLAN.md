    SHEBANG + DOCSTRING:
      #!/usr/bin/env python3
      Module docstring describing: token-drift check; reads brandbook/tokens.json admin_css_mapping vs rulestead_admin.css Block 1; mirrors check_synced_pair.py; usage: "python3 scripts/check_brand_tokens.py" from repo root; exit 0 + "BRAND TOKENS SYNCED (N tokens)" on success; exit 1 on mismatch.

    IMPORTS: sys, re, json (stdlib only — no third-party packages)

    CONSTANTS:
      TOKENS_JSON = "brandbook/tokens.json"
      CSS = "rulestead_admin/priv/static/css/rulestead_admin.css"

    FUNCTION extract_css_decls(css, sel) → dict or None:
      Mirrors decls() from check_synced_pair.py but returns a DICT (not sorted list) so per-token diff is possible.
      Algorithm:
        1. i = css.find(sel) — find selector in COMMENT-STRIPPED css (comment strip happens in main, before calling this)
        2. If i < 0: return None
        3. j = css.find("{", i) — find opening brace
        4. Brace-depth walk: k=j, depth=0, while k<len(css): increment depth on {, decrement on }, break when depth==0
        5. Iterate css[j+1:k].splitlines(), strip each line
        6. For each line starting with "--rs-" and containing ":": partition on ":", result[name.strip()] = val.strip().rstrip(";")
        7. Return result dict

    FUNCTION main() → int:
      1. tokens = json.load(open(TOKENS_JSON))   # raises JSONDecodeError on malformed JSON — free validity check
      2. mapping = tokens["admin_css_mapping"]["light"]
      3. raw = open(CSS).read()
      4. css = re.sub(r"/\*.*?\*/", "", raw, flags=re.S)   # MUST happen before any css.find() call (Pitfall 3)
      5. css_decls = extract_css_decls(css, ".rs-shell,")   # locate Block 1 using trailing-comma form
      6. If css_decls is None: print("ERROR: Block 1 selector '.rs-shell,' not found in CSS"); return 1
      7. mismatches = [], matched = 0
      8. For name, expected in sorted(mapping.items()):
           css_val = css_decls.get(name)
           If css_val is None: mismatches.append(f"  {name}: tokens.json={expected}  css=<missing>")
           Elif css_val.lower() != expected.lower():  # case-insensitive: #3A6F8F == #3a6f8f
             mismatches.append(f"  {name}: tokens.json={expected}  css={css_val}")
           Else: matched += 1
      9. If not mismatches: print(f"BRAND TOKENS SYNCED ({matched} tokens)"); return 0
      10. print("BRAND TOKEN DRIFT DETECTED")
          For m in mismatches: print(m)
          return 1

    ENTRY POINT:
      if __name__ == "__main__": sys.exit(main())

    POST-WRITE: chmod +x scripts/check_brand_tokens.py

    CRITICAL PITFALL (Pitfall 3): comment stripping MUST occur in step 4 BEFORE any css.find() call. The rulestead_admin.css header comment (~lines 167–170) documents all four block selectors. Without stripping, css.find(".rs-shell,") matches inside the comment header and the brace-walk starts at the wrong position.
