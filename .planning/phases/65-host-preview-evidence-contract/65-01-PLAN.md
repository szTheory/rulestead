Create `rulestead/lib/rulestead/targeting/preview_evidence.ex`:

```elixir
defmodule Rulestead.Targeting.PreviewEvidence do
  @moduledoc false

  alias Rulestead.Targeting.PreviewEvidence.Query

  @type result ::
          map()
          | keyword()
          | {:ok, map() | keyword()}
          | {:error, atom() | String.t()}

  @callback resolve(query :: map()) :: result()

  @spec resolve(map() | keyword(), keyword()) :: {:ok, map()} | {:error, Rulestead.Error.t()}
  def resolve(query, opts \\ []) do
    query = Query.new(query)

    case resolver_module(opts) do
      nil -> {:ok, %{}}
      resolver when is_atom(resolver) ->
        if function_exported?(resolver, :resolve, 1) do
          normalize_resolver_result(apply(resolver, :resolve, [query]))
        else
          resolver_failed(:invalid_resolver)
        end
    end
  rescue
    _ -> resolver_failed(:invalid_provider_response)
  end

  @spec resolver_module(keyword()) :: module() | nil
  def resolver_module(opts \\ []) do
    Keyword.get(opts, :resolver) ||
      Application.get_env(:rulestead, :preview_evidence_resolver)
  end
end
```

Create `rulestead/lib/rulestead/targeting/preview_evidence/limits.ex`:

Constants (module attributes):
- `@max_sample_rows 25`
- `@max_payload_bytes 16_384` (term byte size after `:erlang.term_to_binary/1` on normalized evidence map)

Impression allowlist keys (strings): `window_label`, `sampled_impressions`, `matched_impressions`, `variant_breakdown`.

Public functions:

**`merge_samples(command_samples, resolver_samples, opts \\ [])`**
- Union explicit command rows first, then resolver rows
- Dedupe by `{actor_key, targeting_key}` when both present; else keep both if keys missing
- Truncate to `Keyword.get(opts, :max_sample_rows, @max_sample_rows)`

**`validate_and_redact(evidence_map, opts \\ [])`**
- Input map keys: `samples` (list), `impression_summary` (map or nil)
- Redact each sample via `Rulestead.Admin.Redaction.redact_metadata/2` with `ImpactPreview` sample allowlist (import or duplicate `@sample_allowlist` reference)
- Redact impression via allowlist-only map; strip PII field names (`email`, `ip`, `user_agent`, `phone`, `name`, `session`, `actor_id`, etc.)
- `variant_breakdown` entries: only `variant` + `count` keys; max 20 entries
- Counts must be non-negative integers
- Reject unknown impression keys → `{:error, invalid_command("preview evidence invalid", metadata: %{code: "preview_evidence_invalid"})}`
- Oversize after normalize → `preview_evidence_oversized`
- `policy_denied: true` in resolver map → `preview_evidence_policy_denied`

**`enforce_payload_size!(map)`** — check byte size; error if over limit

Wire `PreviewEvidence.resolve/2` to pipe `{:ok, raw}` through `Limits.validate_and_redact/2`.

Helper `normalize_resolver_result/1` in facade:
- `{:ok, map}`, bare map, keyword → map
- `{:error, reason}` → `resolver_failed(reason)`
- empty map → `{:ok, %{samples: [], impression_summary: %{}}}`
