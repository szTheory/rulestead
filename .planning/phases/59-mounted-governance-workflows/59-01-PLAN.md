Create `rulestead_admin/lib/rulestead_admin/live/audience_live/governance.ex`:

**Public API:**

```elixir
@spec load_governance_context(Socket.t(), map(), keyword()) :: Socket.t()
# Args: socket, preview map, operation: :update | :archive
```

**Steps inside loader:**
1. `environment_key = socket.assigns.current_environment.key`
2. `protected? = Rulestead.Promotion.Compare.protected_target?(environment_key)`
3. `deps = Rulestead.list_audience_dependencies(Shared.dependency_command(socket, preview.audience_key))` — handle `{:ok, inv}` / `{:error, _}`
4. Compute `hidden_reference_count` from inventory or 0 on error
5. Build assess attrs map from preview + `operation` string + `hidden_reference_count` + dependency entries
6. `Rulestead.assess_audience_blast_radius(attrs)` → assessment or error
7. `visibility_tier` — `:denied` if deps `{:error, :auth}`, `:partial` if hidden > 0, else `:full`
8. `governance_mode`:
   - not protected → `:unrestricted`
   - protected + below → `:direct_apply`
   - protected + above → `:change_request`
   - protected + indeterminate OR assess error OR visibility_tier in [:partial, :denied] → `:blocked`
9. Assign: `:governance_mode`, `:visibility_tier`, `:blast_radius_assessment`, `:dependency_inventory` (normalized map for tables), `:governance_blocked_reason` (top breach remediation string or nil)

Export pure helpers for tests: `governance_mode/3`, `visibility_tier/1`.

**Also add** `approval_expectation_assigns/2` for confirm (used in 59-03):
- Call `Rulestead.Admin.Authorizer.approval_requirement(actor, :submit_change_request, %{resource_type: "audience", resource_key: key}, env_key)`
- Return map with `required_approvals`, `self_approval_allowed?`, `can_submit?` from `Authorizer.authorized?/4`
