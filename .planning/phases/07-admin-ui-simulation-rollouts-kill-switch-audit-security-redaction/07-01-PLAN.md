| Boundary | Description |
|----------|-------------|
| host actor/policy input -> admin mutation facade | Untrusted host session and policy decisions gate writes to runtime-affecting state |
| raw context traits -> telemetry/audit metadata | Potential PII crosses from operator input into emitted/persisted metadata |

| T-07-02 | I | telemetry/audit metadata | mitigate | Add `Rulestead.Admin.Redaction` and require redact-before-emit/persist per `D-33` and `D-34`. |
| T-07-03 | T | audit ledger | mitigate | Preserve append-only semantics and implement rollback as a linked inverse write rather than update/delete. |
| T-07-04 | R | denied production edits | mitigate | Persist denied attempts in the same audit ledger with `result: :denied` and typed auth errors. |
</threat_model>
