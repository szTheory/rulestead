<objective>
Implement the Experimentation Core by embedding an Experiment schema in the Ruleset and supporting a deterministic evaluation strategy based on immutable iteration salts.

Purpose: To allow deterministic A/B testing assignments without stateful cache bloat, ensuring purity of the evaluator and preserving ETS cache performance.
Output: Experiment data model, evaluator support for `:experiment` strategy, telemetry events with experiment metadata, and control plane commands for explicit lifecycle.
</objective>
