| T-06-02 | T | `Rulestead.Flag` lifecycle fields | mitigate | Validate `owner` plus exactly one of `expected_expiration` or `permanent`; reject sentinel-date shortcuts and contradictory lifecycle payloads. |
| T-06-03 | I | stale-state persistence | mitigate | Store `last_evaluated_at` on `flag_environments` only and derive stale state through a classifier so UI screens cannot infer lifecycle state from raw telemetry payloads. |
