  @enforce_keys [:status, :reasons]

Create `rulestead/lib/rulestead/guardrails/auto_advance.ex`:

**API:** `@spec evaluate_eligibility(map(), keyword()) :: {:ok, Eligibility.t()}`

**Policy map keys** (from DB row or test fixture):
- `enabled`, `observation_window_seconds`, `next_stage`, `next_percentage`, `flag_key`, `environment_key`, `rule_key`

**Evaluate opts / attrs** (second arg map or keyword merged):
- `signal_facts` â€” list of maps
- `monitoring_window_ends_at` â€” DateTime or nil
- `evaluated_at` â€” default `DateTime.utc_now() |> DateTime.truncate(:second)`

**Algorithm:**
1. If policy `enabled` is false â†’ blocked, `["auto_advance_disabled"]`
2. If enabled but any of `observation_window_seconds`, `next_stage`, `next_percentage` nil/blank â†’ blocked, `["auto_advance_policy_incomplete"]`
3. If `monitoring_window_ends_at` is nil â†’ blocked, `["monitoring_window_unset"]`
4. Call `Decision.evaluate(signal_facts, evaluated_at: evaluated_at, monitoring_window_ends_at: ends_at)`
5. If not `decision.monitoring_window_closed?` â†’ blocked, `["monitoring_window_active"]`
6. If `signal_facts == []` and closed â†’ blocked, `["monitoring_window_expired"]`
7. If `decision.state != :healthy` â†’ blocked with reason `"guardrail_#{state}:#{decision.reason}"` (e.g. `guardrail_held:stale`, `guardrail_pending_data:monitoring_window_active`, `guardrail_rollback_triggered:breached`)
8. Else â†’ eligible, `reasons: []`

Populate `policy_snapshot` with normalized policy fields (no secrets). Populate `decision_summary` from decision struct.

Zero I/O â€” no Repo, no GenServer.
