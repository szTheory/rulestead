    rulestead is the runtime package in the Rulestead sibling-package release.

    Use this package when your application needs deterministic flag evaluation, typed values, context builders, installer support, and fake-backed test helpers without mounting the admin UI.

    Install

    The first public Hex release for rulestead is planned for after v0.6.0.

    defp deps do
  [
    {:rulestead, "~> 0.1"}
  ]
end
    mix deps.get
mix rulestead.install
mix ecto.migrate
